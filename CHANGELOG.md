# Changelog - Arianne-Support

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-20
### Added
- Initial Open Source release of the Arianne-Support Printer Deployment System.
- Core engine `A-P2P.ps1` for professional printer deployment and diagnostics.
- Unified entry point `Patch.bat` for end-user execution.
- External configuration dictionary `VIP_Config.json`.
- Full diagnostic suite for system integrity checks.
- Internal logic for deep uninstallation (Rollback) of system drivers via PnPUtil.
- Bilingual documentation (French/English).
- GitHub ecosystem with CI (Linting) and Release (Packaging) workflows.
